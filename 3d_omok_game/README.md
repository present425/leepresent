# 🎮 3D 오목 게임

Python과 Pygame을 사용하여 구현한 3D 효과가 있는 오목 게임입니다.

## ✨ 주요 기능

- **완전한 오목 규칙 구현**: 가로, 세로, 대각선으로 5개 돌을 연속으로 놓으면 승리
- **2인용 모드**: 두 명의 플레이어가 번갈아가며 게임
- **AI 대전 모드**: 미니맥스 알고리즘을 사용한 지능적인 AI와 대전
- **3D 시각 효과**: 입체적인 보드판과 돌의 그림자 효과
- **직관적인 UI**: 현재 플레이어 표시, 게임 상태 안내

## 🎯 게임 규칙

1. 흑돌(검은색)이 먼저 시작합니다
2. 가로, 세로, 대각선으로 5개 돌을 연속으로 놓으면 승리합니다
3. 보드가 가득 차면 무승부입니다

## 🚀 설치 및 실행

### 1. 필요한 패키지 설치

```bash
pip install -r requirements.txt
```

### 2. 게임 실행

```bash
python main.py
```

## 🎮 조작법

- **마우스 클릭**: 돌을 놓을 위치 선택
- **ESC**: 게임 종료
- **R**: 게임 재시작
- **1**: 2인용 모드로 전환
- **2**: AI 대전 모드로 전환
- **E**: AI 난이도 쉬움으로 설정
- **M**: AI 난이도 보통으로 설정
- **H**: AI 난이도 어려움으로 설정

## 🧠 AI 알고리즘

### 난이도별 AI 동작

1. **쉬움 (Easy)**: 랜덤 수 선택
2. **보통 (Medium)**: 
   - 즉시 승리할 수 있는 수 우선 선택
   - 상대방의 승리 수 차단
   - 중앙 근처 우선 배치
3. **어려움 (Hard)**: 
   - 미니맥스 알고리즘 사용
   - 알파-베타 가지치기로 성능 최적화
   - 3단계 깊이까지 탐색

## 🎨 3D 효과

- **입체 보드판**: 다층 그림자와 하이라이트로 진짜 나무 보드 같은 입체감
- **나무 질감**: 실제 나무 보드의 질감을 시뮬레이션한 텍스처 효과
- **3D 돌**: 조명 효과, 그림자, 하이라이트, 반사광이 있는 입체적인 돌
- **돌 놓기 애니메이션**: 돌이 위에서 떨어지는 자연스러운 애니메이션
- **마우스 호버 효과**: 마우스를 올리면 깜빡이는 파란색 링으로 위치 표시
- **배경 그라데이션**: 위에서 아래로 어두워지는 자연스러운 배경
- **승리 라인 3D**: 그림자와 하이라이트가 있는 황금색 승리 라인
- **UI 반투명 효과**: 나무 질감의 반투명 UI 패널
- **조명 시스템**: 화면 상단에서 비치는 조명으로 입체감 강화

## 🔧 기술 스택

- **Python 3.8+**: 메인 프로그래밍 언어
- **Pygame 2.5.2**: 게임 엔진 및 그래픽 라이브러리
- **NumPy 1.24.3**: 수치 계산 및 배열 처리
